How we fixed Miracast issue with USB WIFI dongle

By tinyDebian
Tags: embedded Linux USB 5G Wi-Fi

Description of the issue:

Before InFocus launched LightCast, Miracast did not reliably work with some Android phones and Windows PC.

There are a lot of models with different Android version and different WiFi chipset and a lot of models of Windows PC with diffrent WiFi chipset in the market. Compatibility is always a huge challenge.

Link to InFocus LightCast

Challenge:

Compatibility is always a huge challenge.

Our approach:

By fully understanding WiFi Direct and Miracast specification, good understanding of RTL8812AU driver, Android wpa_supplicant, sophisticated debug technique and a lot of testing, we made several changes and got InFocus LightCast reliably working at an acceptable level.
